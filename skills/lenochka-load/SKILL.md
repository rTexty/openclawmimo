# lenochka-load — Подгрузка истории чатов

## Использование

1. Написать `/load` в личном чате с ботом
2. Отправить HTML или JSON файлы (дампы чатов из Telegram)
3. Нажать кнопку "Готово"
4. Леночка обработает файлы и запишет сообщения в БД

## Поток

```
/load → 📎 Жду файлы
[файл] → 📎 Получен: filename
Готово → ✅ Загружено N сообщений от M контактов
```

## Парсеры

- `parsers/html_parser.py` — парсит Telegram HTML dump
- `parsers/json_parser.py` — парсит Telegram JSON export
- `run_load.py` — оркестратор импорта

## Ограничения

- Только личный чат с ботом (`@lenochkab2b_bot`)
- Только owner может использовать
- Файлы: HTML, JSON
- Сообщения сохраняются как `chit-chat` (без LLM классификации)
- Для каждого уникального отправителя создаётся отдельный контакт и chat_thread
