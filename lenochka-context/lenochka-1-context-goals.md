# Lenochka – контекст, цели, крайние случаи

## Визия продукта

Lenochka — персональный AI-ассистент в Telegram, который работает как невидимая CRM/операционный помощник поверх мессенджера.

Ключевая идея: пользователь продолжает жить в Telegram, а Lenochka незаметно превращает поток сообщений в структурированные сущности (лиды, сделки, задачи, договорённости, документы), следит за «хвостами» и помогает не терять деньги и время.

## Основные сценарии

1. Коммуникации
2. Задачи
3. Лиды и продажи
4. Документы и финансы
5. База знаний и аналитика

## Краевые случаи

1. Смешанные чаты (личное + рабочее)
2. Мультичат с одним клиентом
3. Изменение решений (supersede/override)
4. Ошибки извлечения (ложные лиды)
5. Безопасность и приватность
6. Мультиюзер и права

## Цели первого релиза

1. Ядро: интеграция с Telegram, нормализация сообщений, CRM-схема, классификатор
2. Память: Agent Memory Ultimate + CHAOS Memory
3. Skills: stream-triage, reflection-briefings, dynamic-context
4. UX: утренний дайджест, команды /status, /leads, /tasks

## Метрики успеха

- % лидов, дошедших до «обработан»
- Кол-во брошенных диалогов
- Среднее время от лида до первого ответа
- Кол-во задач done
- NPS/CSAT
